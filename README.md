# peakwarden-dashboard -- Title: "Cockpit"
# What is Cockpit?
Cockpit is a custom-built personal intelligence dashboard designed to deliver a variety of curated news sources across 22 categories -- from geopolitics and AI industry news to pop culture and local events. Built as a standalone Progressive Web App and installable directly to your desktop, Cockpit stores, organizes, and allows immediate access to your most valued online information sources, while cutting the time taken to search each individual source through your browser. Think of it as a personalized launchpad for staying informed on your terms.
# Editorial Philosophy:
Every source in Cockpit is there because you chose it. For me, each source is held to a single standard: Tasteful, data-driven, actionable, and mentally satiating - the kind you feel smarter for having read; news worth discussing over coffee. No empty headlines, no wasted time.
# Features:
- 🏔️ Full-screen mountain background — alpine clarity as a design metaphor
- 🃏 22 frosted glass category cards in a 5x4 grid
- ✨ Apple-style card animations — hover bounce, click-to-expand panel
- 🔗 60+ hand-curated sources across every major intelligence category
- 📱 Installable as a PWA — runs as a standalone desktop app
- 🌐 Live on the web via GitHub Pages

# Install as PWA:
1. Open the live link in Chrome
2. Click the install icon in the address bar (small monitor icon with a download arrow)
3. Click **Install** in the prompt
4. Cockpit opens as a standalone desktop app with its own taskbar icon
# Tech Stack:
- HTML / CSS / JavaScript — single file architecture
- GitHub Pages — free hosting and deployment
- PWA — manifest.json + service worker for installability

# As A Developmental Model
Cockpit aestablishes a reusable architecture for future dashboard projects:
- Proven design language — dark frosted glass, full background image, grid layout
- Established deployment pipeline — GitHub → GitHub Pages → PWA
- Scalable card system — any category set, any link collection
- Claude Code compatible — prompt-driven development workflow
- Installable as standalone app
- Integratahble into larger dashboard systems
Future dashboards can fork this model for project management, financial tracking, creative workflows, or any domain requiring a personal command center.
